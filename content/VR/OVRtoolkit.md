## はじめに  
今回はSteamVRで便利なソフト「OVR Toolkit」を紹介します。  

### 何ができるソフトなのか  
このソフトはVR内でデスクトップを表示、操作を行うことができます。  
ブラウジングなど 簡単な事なら難なくこなせます。  

## 導入手順  
[こちらのSteamストアページ](https://store.steampowered.com/app/1068820/OVR_Toolkit/)から購入できます。  
インストールと起動までは他のSteamゲーム等と同じです。  

### SteamVR起動時に同時起動について  
SteamVRの設定でVR起動時に同時起動ができるのですが、僕の環境では動作が不安定なのでおすすめしません。  
具体的には ホームメニューが表示されません（トラッカー等に吸われてる？）  
コントローラ、トラッカー等を全て起動した後に本ソフトを起動することを推奨します。  

## VR内での設定  

### メニューの出し方  
OVRTookKitを起動すると デスクトップの方に下記画像のようなウィンドウが表示されています。  

![DesktopImage](/images/VR/OVRToolkit/DesktopImage.png)  

ValveIndexではこのように表示されていると思います。  
この画像に「両方のグリップボタンを同時に2回押します」と書いています。  
ちなみに、Indexでのグリップボタンとはコントローラをぎゅっと握ることです。  

それぞれのコントローラで操作方法が異なると思うので 各自で確認してください。  

それでは、実際にVR内での設定を紹介していきます。  

### 手首オーバーレイ  
本ソフトを起動すると 手首に下画像のようなUIが出てきます。  

![TekubiUI](/images/VR/OVRToolkit/OverView.png)  

現在時刻、VR起動時間、バッテリー残量が見れる優秀なUIです。  

### メインメニュー  
VR内でメニューを出すと一番最初に出てくる画面です。  

![MainMenu](/images/VR/OVRToolkit/Mainmenu.png)  

下部メニューから各設定やキーボードを開けます。  
中央下部の「新しいウィンドウを作成する」から新規ウィンドウを開けます。  

### 新規ウィンドウを開くと  
新規ウィンドウを開くと 近くにウィンドウがドロップします。  
こちらはグリップ長押しで 掴んで移動させる事ができます。  

### 全体設定  
メニュー下部の「全体設定」を押すと出てくる画面です。  

![GlobalSetting1](/images/VR/OVRToolkit/Setting1.png)  
![GlobalSetting2](/images/VR/OVRToolkit/Setting2.png)  

- # 手首のオーバーレイを有効にする  
上で記した 手首に表示されるUIの表示 非表示を切り替えます。  
邪魔な場合はオフにしましょう。  

- # CPUおよびRAM情報を表示  
手首のオーバーレイにCPU.RAM.GPUの使用率を追加します。  
使用率が気になる場合はオンにしましょう。  

- # オーバーレイハンドの切り替え  
手首オーバーレイの表示される手が切り替わります。  

- # タッチスクリーン入力を使用する  
__要検証__  

- # タッチパッドのスクロールを許可する  
タッチパッドを使用して画面スクロールできるようになります。  
これを使用しない場合は トリガーで掴んで離すという動作でスクロールできます。  

- # グリップウィンドウのサイズ変更を使用する  
__要検証__  

- # 回転ロックを有効にする  
__要検証__

- # 開いているメニューを最小化  
__要検証__

- # コンパクトUIを使用する  
メインメニューが小さくなります。  
設定等を開くと結局大きくなるのでホットキーを使用しない方はオフ推奨です。  

### ウィンドウ設定  
メニュー下部のウィンドウ設定を押すと出てくる画面です。  
VR内で表示している各ウィンドウをクリックして 設定ウィンドウを指定します。  

![WindowSetting1](/images/VR/OVRToolkit/WindowSetting1.png)  
![WindowSetting2](/images/VR/OVRToolkit/WindowSetting2.png)  

- # ピンウィンドウ  
ここにチェックを入れるとメニューを閉じた後も ウィンドウが表示されます。  
初期設定だとメニューを開いてる間しかウィンドウが表示されないので 要確認項目です。  

- # ブロックカーソル入力　　
__要検証__  

- # 不透明度  
減少させるほど透明に近づきます。  
初期値は100です。減少させればウィンドウの後ろが見えるようになりますが、同時に視認性も落ちていきます。  

- # フレームレート  
画面のフレームレートを設定できます。  
数値が高いほどぬるぬる動きます。  

- # エコモードを有効にする  
有効にすると 見ていない（画面中央付近で画面を捉えていない）時にフレームレートが大幅に落とされます。  
動画など 常に画面が動き続ける場合はオフを推奨します。  

- # ウィンドウを見ない時に隠す  
有効にすると 見ていない時に画面が消えます。ウィンドウをおいている場所を見ると表示されます。  

- # 遺跡対象デバイス  
どのデバイスに追従するかを設定できます。  
無しの場合はリアル座標、左右の手はコントローラに  
HMDだと頭に追従されます。
